Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


      def fahrenheit(n)
          return (n * 9/5) + 32
      end


      puts "Enter degrees in Celsius"
      Celsius = gets.chomp

      Var1 = Celsius
      Var2 = Var1.to_f
      Fahrenheit = fahrenheit(Var2)

      reply = "#{Var2} degrees Celsius is
              #{Fahrenheit} degrees Fahrenheit"
      puts reply

Ordinal Challenge

This Ruby program will convert a plain number to the ordinal of the number. For example, if the user enters 2, it will display 2nd, if the user enters 3, it will display 3rd, etc. 


    def suffix (number)
        right_most_digit = number % 10
        
        if right_most_digit == 1 && number != 11
            return "st"    
        elsif right_most_digit == 2 && number != 12
            return "nd"
        elsif right_most_digit == 3 && number != 13
            return "rd"
        else
            return "th"
        end
        
    end

    puts "Enter a number"
    number = gets.chomp.to_i

    puts "That's the #{number}#{suffix(number)} item!"

Foobar

This Ruby program is an adaption of a classic technical interview problem which displays sequences of the Foobar pattern. 


    def foobar(number)
        number_divisible_by_5 = number % 5
        number_divisible_by_3 = number % 3
        
        if number_divisible_by_5 == 0 && 
           number_divisible_by_3 == 0
            return "Foobar"
        elsif number_divisible_by_5 == 0
            return "Bar" 
        elsif number_divisible_by_3 == 0
            return "Foo"
        else
            return number
        end
    end

    puts "How many items do you want to see?"
    number = gets.chomp.to_i

    current = 1

    while current <= number
        puts "#{foobar(current)}"
        current = current + 1
    end
